About This Tool

Scam Message Checker analyzes suspicious SMS messages, WhatsApp messages, emails, and links. It gives a risk score, explains suspicious wording, extracts URLs, and shows whether the link should be opened.

When hosted on Cloudflare Pages with Functions, it can check redirect chains, final destination URLs, HTTPS status, HTTP status codes, and security headers. GitHub Pages runs only the static browser-side fallback mode.

This tool is designed for awareness and defensive checking. It cannot guarantee that a link is fully safe.
